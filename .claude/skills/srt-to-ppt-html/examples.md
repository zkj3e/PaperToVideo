```
用 srt-to-ppt-html：把 斯坦福重磅实证报告：AI怎么落地？_zh-cn.srt 做成报纸风的 PPT HTML，12 页
```

```
用 srt-to-ppt-html：把 @input.srt 转成赛博朋克风的 HTML 幻灯片
```

## 示例输出结构（报纸风，12 页）

以下展示从斯坦福报告 SRT 生成 HTML 时的分页逻辑和各页使用的组件。**每页均记录对应字幕的时间范围。**

### 第 1 页 · 封面
- 组件：`.kicker` + `.display` + `.title.accent`
- 时间范围：`00:00 – 00:07`（封面页使用第一条字幕的时间）
- 内容：标题「斯坦福重磅实证报告：企业AI实施手册」+ 副标题「不是预测推演，而是基于51个真实案例的落地指南」

### 第 2 页 · 报告背景
- 组件：`.headline` + `.body`
- 时间范围：`00:07 – 00:22`
- 内容：报告调研范围、回答的核心问题

### 第 3 页 · 数据对比（落地阻力）
- 组件：`.stat-card` × 2 + `.quote`
- 时间范围：`00:22 – 00:40`
- 内容：77% 项目最难挑战与技术无关 vs 法务/HR/风控合规成最大阻力

### 第 4 页 · 案例（组织变革）
- 组件：`.headline` + `.body` + `.card`
- 内容：制造业公司把 AI 采用率写进 OKR，法务从阻碍者变推动者

### 第 5 页 · 数据对比（容错）
- 组件：`.stat-card` × 2
- 内容：61% 项目至少失败过一回 vs 73% 项目从小规模试点开始

### 第 6 页 · 隐形成本
- 组件：`.headline` + `.quote` + `.body`
- 内容：100 万 AI 预算，背后流程重构、培训、管理转型成本可能 10 倍

### 第 7 页 · 人机协作模式
- 组件：`.stat-card` × 2 + `.tag-row`
- 内容：升级模式 71% vs 审批模式 30%，效率差 2.4 倍

### 第 8 页 · 案例（食品平台）
- 组件：`.headline` + `.stat-number` + `.body`
- 内容：AI 处理 90-95% 订单问题，从 1500 单飙到 4 万单，团队仅 1.5 人

### 第 9 页 · 安全与数据
- 组件：`.headline` + `.quote` + `.tag-row`
- 内容：安全是护城河而非阻碍 + 15% 影子AI风险 + 数据不需要完美

### 第 10 页 · AI 增收模式
- 组件：`.headline` + `.tag-row` + `.card`
- 内容：个性化转化（购买率 +20%）、速度变武器（周→4小时）、内部工具变产品

### 第 11 页 · 降本与裁员
- 组件：`.stat-card` + `.quote` + `.body`
- 内容：45% 减人头 vs 55% 其他（加速/转变任务/裁员），战略选择决定结果

### 第 12 页 · 收束
- 组件：`.display` + `.quote` + `.tag-row`
- 内容：「AI 变革的窗口正在关闭，不是因为技术不成熟，而是因为竞争对手已经在行动」

### 校验
- 12 页 ✓
- 每页有 footer 页码 ✓
- 每页有 `data-time-start` / `data-time-end` 属性 ✓
- 每页 footer 显示时间范围 ✓
- 数字全部忠实于原字幕 ✓

## 赛博朋克风示例（8 页）

赛博朋克风更紧凑，适合核心数据展示：

### 第 1 页 · 封面
- 组件：`.kicker` + `.display` + `.subtitle`
- 时间范围：封面页，使用第一条字幕时间
- 内容：英文主标题 + 中文副标题

### 第 2 页 · 核心数据
- 组件：`.panel` > `.stat` > `.number` + `.label`
- 时间范围：对应字幕段落起止时间
- 内容：94% 理论覆盖 vs 33% 实际暴露

### 第 3–7 页 · 关键发现
- 组件：`.subtitle` + `.panel`
- 每页一个核心发现 + 支撑数据

### 第 8 页 · 结尾
- 组件：`.display` + `.subtitle`
- 内容：金句收尾

## 页面布局选择指南

| 内容特征 | 报纸风组件 | 赛博朋克组件 |
|----------|-----------|-------------|
| 封面/章节标题 | `.display` / `.headline` | `.display` + `.subtitle` |
| 单个大数字 | `.stat-card` > `.stat-number` | `.panel` > `.stat` > `.number` |
| 两个对比数字 | 两个 `.col-6` `.stat-card` | `.panel` > `.stat`（两个 `.number`） |
| 金句/观点 | `.quote` | `.subtitle`（大字号） |
| 职业/标签列表 | `.tag-row` > `.tag` / `.tag.red` | `.panel` + 内联标签 |
| 补充说明 | `.card` + `.small.sans.muted` | `.panel`（统一风格） |
| 正文段落 | `.body` | `.subtitle` |
| 分类标题 | `.kicker` | `.kicker` |
